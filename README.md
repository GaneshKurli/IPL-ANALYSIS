KEY FEATURES

Interactive Dashboards:
Multiple dashboards for Team Performance, Player Statistics, Match Summary, and Stadium Analysis — all built with user-friendly filters and tooltips.

Dynamic Filters:
Easily explore data by Year, Team, Player, and Stadium using interactive filters and parameters.

Custom Calculations:
Created calculated fields to measure key cricket metrics:

Strike Rate = (Runs / Balls) × 100

Economy Rate = Runs Conceded / Overs Bowled

Win Percentage = (Matches Won / Matches Played) × 100

Comparative Analysis:
Compare teams and players across seasons using parameter-based controls and trend charts.

Data Storytelling:
Built clean, colorful visuals that explain patterns in team wins, player impact, and seasonal trends — easy to understand at a glance.

Visual Highlights:

Line chart for Season-wise team wins

Bar chart for Top 10 batsmen by strike rate

Pie chart for Match results by stadium

Table for Team-wise performance comparison

Clean Design:
Used Tableau formatting best practices — consistent colors, readable labels, and intuitive dashboard layouts.

 Tool Used: Tableau Desktop
 Dataset: IPL Match & Player Statistics (CSV)
